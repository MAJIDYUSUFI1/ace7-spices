# Ace7 Spices — Image Prompts for Nano Banana

Every image on the site is a real `<img>` pointing to a file inside the `images/` folder. Until the file exists, the site shows a tasteful labelled placeholder, so it already looks finished. Generate each image below, **save it with the exact filename shown**, drop it into `/images`, and it appears automatically — no code changes needed.

> Tip: keep filenames and extensions exactly as written (all `.jpg`). If you export `.png` or `.webp`, either rename to `.jpg` or update the `src` in the HTML.

---

## How to use this

1. Paste the **Shared style line** (below) at the **start** of every prompt — it keeps the whole set visually consistent.
2. Then paste the specific prompt for that filename.
3. Generate at the **aspect ratio** noted next to each (the layout is built for these).
4. Save as the **exact filename**.

### Shared style line (prepend to every prompt)

```
Premium editorial commercial photograph for a high-end Indian spice export brand. Warm, natural, true-to-life color; rich reds, deep turmeric gold and forest greens; soft directional daylight; shallow depth of field; fine realistic detail and texture; no text, no logos, no watermarks, no people looking at camera unless specified. Authentic Indian setting. Photorealistic, shot on a full-frame camera, 50mm lens feel.
```

## A note on consistency
Generating all images in one session with the shared style line keeps lighting and colour consistent. If a generated image comes out too busy where text overlays it (the heroes), regenerate asking for "darker, simpler, more negative space on the left."
